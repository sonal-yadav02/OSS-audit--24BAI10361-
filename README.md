## Project Overview
This project conducts a structured audit of Git, a distributed version control system created by Linus Torvalds in 2005. The audit explores Git's origin story, licensing under GPLv2, ethical implications, Linux footprint, and its role within the broader open-source ecosystem.

## Repository Contents
| File | Description |
|------|-------------|
| `script1_system_identity.sh` | Displays Linux system information (distribution, kernel, user, uptime, license) |
| `script2_package_inspector.sh` | Checks if Git is installed and displays version with description |
| `script3_disk_auditor.sh` | Audits disk usage and permissions of important system directories |
| `script4_log_analyzer.sh` | Analyzes log files for ERROR/WARNING patterns |
| `script5_manifesto_generator.sh` | Generates a personalized open-source manifesto interactively |

## Prerequisites
- Linux operating system (Ubuntu/Debian/CentOS/RHEL)
- Bash shell
- Git installed (for Script 2)
- Read permissions on log files (for Script 4)
